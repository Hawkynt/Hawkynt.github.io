SZ WordPad - Free Online Rich Text Editor & Document Writer

A free browser-based rich text editor with ribbon UI, table editing, multi-format import/export (HTML, RTF, DOCX, TXT, PDF), page layout controls, equation editor, and print-layout document view.

Features

  • Full rich text editing: bold, italic, underline, strikethrough, subscript, superscript, fonts, and colors
  • Paragraph formatting: alignment, lists, indentation, line spacing, borders, shading, and heading styles
  • Table creation and editing with row/column management, cell merging, and styling
  • Multi-format import/export: HTML, RTF, DOCX, TXT, and PDF via print
  • Image, shape, text box, hyperlink, bookmark, and symbol insertion
  • Find and replace with case-sensitive and regex search modes
  • Page layout: size, orientation, margins, watermarks, and background color
  • Print Layout, Web Layout, and Outline views with rulers and navigation pane
  • Equation editor with math symbols, fractions, roots, and Greek letters
  • Ribbon UI with Home, Insert, Page Layout, and View tabs
  • Format Painter, spelling check, and zoom from 25% to 500%
