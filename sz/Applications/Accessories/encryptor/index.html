About Data Encryption

Data Encryption - SZ Desktop Encryptor

A simple XOR-based text encryption and decryption utility for the SynthelicZ desktop. Encode text into hex-encoded ciphertext and decode it back using a user-provided key. Demonstrates basic cryptographic concepts in an accessible interface.

Features

  • XOR-based text encryption with user-provided key
  • Hex-encoded ciphertext output and hex-to-plaintext decryption
  • Input validation with descriptive error messages for malformed hex data
  • Key validation with visual feedback for empty key attempts
  • Clipboard integration to copy output text
  • Swap input and output for efficient encrypt/decrypt round-trips
  • Clear all fields with one click
