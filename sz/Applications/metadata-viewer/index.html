About Metadata Viewer

Metadata Viewer

File Analysis & Metadata Editor

Identifies file types via magic bytes, parses format-specific metadata structures, displays embedded images, computes hashes/checksums, and edits metadata fields.

Supports 30+ formats with 29 hash/checksum algorithms. JPEG, PNG, MP3 (Xing/LAME/VBRI), FLAC, WAV (PCM analysis), OGG, MP4 (H.264/H.265 SPS), MKV, PDF, ZIP, 40+ archive formats, PE, ELF, Mach-O, APK (deep), Java .class, fonts, C64/Amiga disk images, and more.
