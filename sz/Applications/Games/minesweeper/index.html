Minesweeper - Free Classic Mine Sweeper Puzzle Game Online

Play the classic Windows Minesweeper game in your browser with Beginner, Intermediate, Expert, and Custom difficulty levels. Features faithful recreation of the original gameplay with enhanced visual effects, statistics tracking, and SVG icons.

Features

  • Four difficulty levels: Beginner 9x9, Intermediate 16x16, Expert 30x16, and Custom fields
  • Safe first click with 3x3 guaranteed safe zone
  • Left-click reveal, right-click flag, chord reveal mechanics
  • Question mark toggle support for advanced play
  • LED-style mine counter and timer with ghost digit display
  • Iconic smiley button with state changes: happy, surprised, cool, dead, skull
  • Difficulty-themed color palettes: meadow green, steel blue-gray, inferno red
  • Canvas particle explosions, chain detonation, confetti win celebration
  • Best times leaderboard and game statistics with win streaks
  • Cinematic multi-phase death animation with skull transition
  • Cross-platform SVG icons for consistent rendering
  • Resizable window with aspect-ratio scaling
