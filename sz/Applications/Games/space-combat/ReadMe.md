# Space Combat

A top-down space shooter for the SynthelicZ Desktop. Pilot your ship in intense dogfights against waves of enemies, upgrade weapons, and chase high scores with explosive visual effects.

## User Stories

### Core Gameplay
- [x] As a player, I can pilot a spaceship with 8-directional movement using WASD/Arrow keys so that I have full control over my position
- [x] As a player, I can fire weapons with Space/Z so that I can destroy enemies
- [x] As a player, I can face escalating waves of enemies with increasing difficulty so that the game stays challenging
- [x] As a player, I can earn a score multiplier for consecutive kills within 2 seconds (up to 12x) so that skilled play is rewarded
- [x] As a player, I can start with 3 lives and lose one when hit with shield down so that I have limited chances

### Weapons
- [x] As a player, I can use 9 weapon types (Spread Shot, Laser, Missiles, Plasma, Beam, Shotgun, Lightning, Flak, Railgun) so that combat has variety
- [x] As a player, I can collect weapon pickups dropped by destroyed enemies so that my firepower improves
- [x] As a player, I can experience per-weapon cooldown rates so that each weapon plays differently
- [x] As a player, I can use homing missiles that track enemies so that I can hit evasive targets
- [x] As a player, I can use the plasma weapon for area-of-effect damage so that I can handle groups
- [x] As a player, I can use the lightning weapon that chains between nearby enemies so that clustered enemies are punished
- [x] As a player, I can use the shotgun for a spread of short-range pellets so that close encounters are rewarding
- [x] As a player, I can use the railgun for high-damage piercing shots so that I can take out tough targets
- [x] As a player, I can use the flak cannon that detonates into fragments so that I can cover wide areas

### Enemies
- [x] As a player, I can face 11 enemy types (Scout, Fighter, Dart, Bomber, Swarm, Sniper, Cloaker, Carrier, Minelayer, Juggernaut, Shieldbearer) so that each wave feels different
- [x] As a player, I can see enemies with distinct behaviors (weaving, shooting back, dropping bombs, cloaking, deploying mines, projecting shields) so that I must adapt my strategy
- [x] As a player, I can face swarm enemies that appear in groups so that I must handle crowd situations

### Boss Fights
- [x] As a player, I can face boss fights every 5 waves so that I have epic confrontations
- [x] As a player, I can fight 4 boss types (Commander, Destroyer, Mothership, Dreadnought) with scaling difficulty so that bosses stay challenging
- [x] As a player, I can see bosses with regenerating shields so that I must sustain damage output
- [x] As a player, I can face multiple boss attack patterns (aimed bursts, sweep arcs, spiral bursts, laser beams, missile salvos, cross beams, mine deployment, bullet rings, minion spawning) so that each fight requires different dodging
- [x] As a player, I can see a boss health bar at the top of the screen so that I know how much damage is left to deal

### Upgrades
- [x] As a player, I can collect persistent upgrade pickups (Multi-shot, Damage, Speed, Fire Rate, Shield, Bullet Size, Magnet Range) so that my ship grows stronger over the run
- [x] As a player, I can upgrade each stat up to 5 levels so that there is meaningful progression

### Shield System
- [x] As a player, I can start with 3 shield HP that absorbs hits so that I have protection before losing a life
- [x] As a player, I can see a shield shimmer visual effect when hit so that I know the shield absorbed damage

### Visual Effects
- [x] As a player, I can see thruster trail particles when moving so that speed is visually communicated
- [x] As a player, I can see explosion particles when enemies and players are destroyed so that destruction feels impactful
- [x] As a player, I can see screen shake on player/enemy hits so that impacts feel visceral
- [x] As a player, I can see floating score text on enemy kills so that I get immediate point feedback
- [x] As a player, I can see laser glow and projectile effects so that weapons look visually distinct

### Game Management
- [x] As a player, I can see a tutorial overlay on first play that teaches controls and mechanics so that I can learn quickly
- [x] As a player, I can toggle the tutorial with H key so that I can review it anytime
- [x] As a player, I can have high scores persisted to localStorage so that I can track my best runs
- [x] As a player, I can pause/resume with Escape so that I can take breaks
- [x] As a player, I can start a new game with F2 so that I can restart anytime
- [x] As a player, I can use touch controls (drag to move, tap to fire) so that the game works on mobile devices

### Integration
- [x] As a player, I can see a menu bar with Game (New, Pause, High Scores, Exit) and Help (How to Play, Controls, About) so that all actions are discoverable
- [x] As a player, I can use dialog windows for high scores, controls, and about info so that information is organized

### Planned Features
- [ ] As a player, I can hear sound effects and background music so that the game has audio atmosphere
- [ ] As a player, I can unlock new ships with different stats so that I have long-term progression goals
- [ ] As a player, I can play a co-op mode with a second player so that I can team up with a friend
- [ ] As a player, I can face environmental hazards like asteroid fields and nebula clouds so that the battlefield has more variety
- [ ] As a player, I can see an achievement system tracking milestones so that I have secondary objectives

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move ship (8 directions) |
| Space / Z | Fire weapon |
| Escape | Pause / Resume |
| F2 | New Game |
| H | Toggle tutorial overlay |

Touch: Drag to move, tap to fire.
