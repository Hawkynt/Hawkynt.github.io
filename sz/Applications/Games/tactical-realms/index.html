Tactical Realms - Tactical RPG / Dungeon Crawler


A grid-based tactical RPG with Shining Force-style combat, D&D Forgotten Realms setting, procedurally generated dungeons, daily content rotation, and mouse-only play.


Features


    

  • Grid-based tactical combat with terrain bonuses and flanking
    • 

  • Procedurally generated overworld and dungeon maps
    • 

  • 8 D&D races and 8 classes with d20 mechanics
    • 

  • Daily character roster rotation and seasonal content
    • 

  • Encrypted save system with auto-save
    • 

  • Mouse-only input for accessibility
    •
