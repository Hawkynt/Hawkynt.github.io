# Platformer

A side-scrolling platformer for the SynthelicZ Desktop. Run, jump, collect coins, stomp enemies, grab power-ups, and defeat bosses across procedurally generated levels with scrolling camera.

## User Stories

- **S-047**: As a player, I want to run, jump, and collect power-ups across side-scrolling platformer levels with enemies, platforms, and boss fights, so that I experience Mario/Sonic-style gameplay.
- **S-048**: As a player, I want platformer to have coin-collect sparkles, power-up activation glow, enemy-defeat particle bursts, and smooth character run/jump animations, so that platforming feels responsive and exciting.
- **S-000**: As a player, I want every game to have particle effects, glow, screen shake, and floating score text, so that games feel polished and exciting.

## Features

- **Side-scrolling** with smooth camera following the player
- **Run and jump** with gravity physics and platform landing
- **Coin collection** with sparkle particles and floating score text
- **Multiple enemy types** — walkers, flyers, and shooters with AI
- **Enemy stomping** — jump on enemies to defeat them with particle burst
- **Power-ups** — double jump, invincibility, fireball with glow aura
- **Boss fights** every 5 levels with health bars and screen shake
- **Level progression** with increasing difficulty (more enemies, longer levels)
- **Lives system** with invincibility on respawn
- **Parallax background** scrolling
- **Tutorial overlay** — 2-page guide shown on first play, toggled with H key, persisted via localStorage
- **High score persistence** via localStorage
- **Menu bar**: Game (New F2, Pause Esc, High Scores, Exit) + Help (How to Play, Controls, About)
- **Dialog system** for high scores, controls, and about

## Controls

| Key | Action |
|-----|--------|
| Arrow Left / A | Move left |
| Arrow Right / D | Move right |
| Arrow Up / W / Space | Jump |
| Z | Fireball (with power-up) |
| Escape | Pause / Resume |
| F2 | New Game |
| H | Toggle tutorial overlay |
