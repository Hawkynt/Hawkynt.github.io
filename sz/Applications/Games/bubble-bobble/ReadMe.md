# Bubble Bobble

A single-screen platformer for the SynthelicZ Desktop. Shoot bubbles to trap enemies, pop them for points, and collect fruit items for bonus score. Features platform physics with screen wrapping, power-ups, and increasing difficulty across levels.

## User Stories

- **S-043**: As a player, I want to control a character that shoots bubbles to trap enemies in a series of single-screen platforming levels with increasing difficulty, so that I experience Bubble Bobble gameplay.
- **S-044**: As a player, I want Bubble Bobble to have bubble pop particle bursts, enemy-trap glow, power-up sparkle effects, and smooth character animations, so that gameplay feels bouncy and fun.
- **S-000**: As a player, I want every game to have particle effects, glow, screen shake, and floating score text, so that games feel polished and exciting.

## Features

- **Bubble shooting** in facing direction with wobble animation
- **Enemy trapping** — bubbles capture enemies with glow effect
- **Pop to defeat** — touch trapped bubbles to pop them with particle burst
- **Fruit item drops** when enemies are popped for bonus score
- **Platform physics** — jump, land on platforms, gravity
- **Screen wrapping** — fall through bottom, appear at top
- **Power-ups** — speed boost, extended range, fire bubbles with sparkle effects
- **Enemy escape** — trapped enemies break free after timeout
- **Level progression** with increasing enemy count and speed
- **Level-clear confetti** particle effects
- **Floating score text** on pops and item collection
- **Screen shake** on player death
- **Lives system** with invincibility on respawn
- **Tutorial overlay** — 2-page guide shown on first play, toggled with H key, persisted via localStorage
- **High score persistence** via localStorage
- **Menu bar**: Game (New F2, Pause Esc, High Scores, Exit) + Help (How to Play, Controls, About)
- **Dialog system** for high scores, controls, and about

## Controls

| Key | Action |
|-----|--------|
| Arrow Left / A | Move left |
| Arrow Right / D | Move right |
| Arrow Up / W | Jump |
| Space / Z / X | Shoot bubble |
| Escape | Pause / Resume |
| F2 | New Game |
| H | Toggle tutorial overlay |
