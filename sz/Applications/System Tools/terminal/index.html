About Terminal

SZ Terminal - Online Command-Line Terminal Emulator with Dual Shell

A dual-shell command-line terminal emulator supporting both cmd.exe and bash shells with piping, redirection, variable expansion, tab completion, batch scripting, and comprehensive built-in commands for the virtual file system.

Features

  • Dual-shell environment with seamless switching between cmd.exe and bash modes
  • File system commands: dir/ls, cd, mkdir, cp/copy, mv/move, rm/del, cat/type, tree
  • Bash text processing: grep, sed, sort, uniq, tr, cut, rev, wc, head, tail, tee
  • Piping, output redirection (>, >>), and command chaining (&&, ||)
  • Environment variable management with expansion syntax for both shells
  • Command history navigation and tab completion for files, directories, and commands
  • Batch file execution with conditional logic (if), loops (for), labels and goto
  • System information commands: ver/uname, systeminfo, ipconfig, tasklist/ps, date/time
  • Aliases (ll, la, l), pushd/popd, du/df/free, seq, cal, and more utilities
