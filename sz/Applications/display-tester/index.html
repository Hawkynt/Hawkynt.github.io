About Display Tester

Display Tester

Comprehensive display quality testing utility with 51 tests across 8 categories: dead pixel detection, color accuracy (banding, dithering), uniformity (sweep, viewing angle, burn-in), FPS/refresh rate, motion/response time (input lag, gray-to-gray, scrolling text), HDR/dynamic range (WebGL2), geometry/sharpness (subpixel layout, moiré), and interactive calibration tools (ruler, pixel clock).

Part of the »SynthelicZ« Desktop Environment.
