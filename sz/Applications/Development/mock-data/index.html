About Mock Data Generator

    

    

  







Mock Data Generator - Realistic Test Data Creator


Generate realistic mock data for testing and development in multiple output formats. Configure custom fields with various data types and export up to 10,000 rows as JSON, CSV, SQL INSERT statements, XML, or TSV.


Features


    

  • Output formats: JSON, CSV, SQL INSERT, XML, and TSV
    • 

  • Configurable field types: names, emails, addresses, phone numbers, dates, UUIDs, and more
    • 

  • Generate up to 10,000 rows of test data
    • 

  • Add, remove, and reorder custom fields
    • 

  • SQL table name configuration for INSERT statements
    • 

  • Syntax-highlighted output preview
    • 

  • Copy generated data to clipboard
    • 

  • Export data as downloadable files
    •
