JSON Viewer - Online JSON Formatter and Validator

View, edit, format, and validate JSON data with a split-pane editor featuring a live tree view. Prettify, minify, sort keys, flatten to dot-notation, and search through keys and values. Copy, open, and save JSON files.

Features

  • Split-pane editor with raw text and interactive tree view
  • Prettify JSON with configurable indentation
  • Minify JSON to remove whitespace
  • Validate JSON syntax with error highlighting
  • Sort keys alphabetically
  • Flatten nested objects to dot-notation and unflatten back
  • Search through keys and values with match navigation
  • Expand and collapse tree nodes
  • Copy all JSON or individual values to clipboard
  • Open and save JSON files
