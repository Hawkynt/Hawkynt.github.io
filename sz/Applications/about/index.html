»SynthelicZ« Desktop

A browser-based desktop environment simulator.

Originally created 2004–2006 by Hawkynt.
Rewritten with modern web technologies.

Visit hawkynt.github.io/sz for the latest version.

🔒 Your data stays local. All files, settings, and application data are stored in your browser’s local storage and persist between sessions — nothing is uploaded to any server. Your saved content will be available every time you return with the same browser on the same device. However, this data is tied to this specific browser and domain. Clearing your browser data, using private/incognito mode, or switching browsers will erase all saved content permanently.
